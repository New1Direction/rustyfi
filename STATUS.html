03

What the machine learned this round

⭐ Contract phase

One canonical API per package

Before bodies are translated, one cheap call pins each package's exact types & signatures — so every file agrees on shapes. Kills the cross-file E0609 / E0308 / E0061 cluster at its source.

⭐ rustfix

The compiler fixes its own errors

200+ errors fixed deterministically, zero tokens — harvesting rustc's machine-applicable suggestions, with MaybeIncorrect guesses kept only if they reduce the count.

The full deterministic spine — no LLM where the compiler already knows the answer

04

The honest read

Where it's genuinely there
  • UX & polish — drag-drop, live progress, honest banner, easter eggs
  • Pipeline robustness — no hangs, lies, or corruption; resumable
  • Pure-logic projects — reliably compile clean (see demo)
  • Trust — cargo is the oracle; it never overstates results
Where the ceiling is real
  • Complex apps — a head-start, not a clean compile
  • Framework semantics — gin → axum needs human/smarter model
  • Run-to-run variance — outputs are a distribution (46–111)
  • No tool clears this — it's the frontier, not a missing pass
